# Video Notes Agent (v0.4.5)

一個基於 AI 的自動化影片筆記與分析系統。

## 🌟 重點功能特色

### 1. 全自動化處理流程
- **YouTube 下載**：支援任何 YouTube 影片連結。
- **AI 語音轉文字**：整合 Whisper 自動產生精準的時間軸字幕。
- **智慧視覺偵測**：系統會根據影片的「場景切換」與「語意停頓」自動擷取關鍵影格（Keyframes），避免雜亂截圖。
- **LLM 多模態分析**：自動為每個影格產生標題、詳細摘要，並進行視覺內容分析。

### 2. 互動式圖書庫 (Library)
- **多維度分類管理**：支援手動建立資料夾與「按日期自動分類」功能。
- **視覺化學習統計**：即時顯示總影片數、已閱讀完食數、已複習次數。
- **學習進度追蹤**：
  - **已看標記**：點擊複選框標記學習進度。
  - **複習計數器**：記錄每部影片的複習次數，幫助強化記憶。

### 3. ⭐ 重點摘錄系統 (Stars)
- **卡片星號收藏**：在單一影片筆記頁面中，點擊影格卡片旁的星號，即可將該片段加入「全體重點」。
- **主題式整理**：支援對星號片段進行二次分類與拖拽排序。
- **快速回溯**：在重點摘錄頁面點擊圖片，可直接跳回影片對應的時間點。

### 4. 強大的編輯與渲染
- **線上及時編輯**：支援直接在網頁上修改 AI 產生的摘要與標題，並自動同步回 YAML 資料庫。
- **自適應渲染系統**：修改資料後自動重新渲染為美觀的 HTML 靜態頁面。
- **左右分欄播放器**：
  - 左側：影片播放器與智慧時間軸（標註關鍵影格、星號片段）。
  - 右側：詳細的影格分析卡片流，與影片播放進度即時聯動。

## 🚀 快速上手

### 1. 系統需求
- Python 3.10+
- `ffmpeg` (需安裝並加入 PATH)
- `yt-dlp` (專案內建或加入 PATH)
- (建議) 本地 Ollama, Gemini 或 Claude API 以進行視覺分析。

### 2. 使用方法
- **啟動圖書庫伺服器**：
  ```bash
  python scripts/server.py
  ```
- **處理新影片**：
  ```bash
  python -m scripts "https://www.youtube.com/watch?v=VIDEO_ID"
  ```
- **更新索引系統**：
  ```bash
  python -m scripts --index
  ```

## 📂 目錄結構
- `library/`：儲存已處理影片的資料（包含 YAML 資料、渲染好的 HTML、以及影格圖檔）。
- `scripts/`：核心實作程式碼，包括下載、分析、渲染與後端伺服器。
